## 🚀 איך להפעיל את הפיילוט

### צעד 1: צור אירוע דמה (2 דקות)

```bash
npm run dev
```

1. היכנס ל-http://localhost:3000/admin
2. לך לטאב "events"
3. לחץ "הוסף אירוע"
4. מלא:
   - שם זוג: "פיילוט Spotify"
   - תאריך: היום
   - מקום: "בדיקה"
5. שמור

### צעד 2: הוסף 10 אורחים (3 דקות)

1. בתוך האירוע, לחץ על טאב "ניהול אורחים"
2. לחץ "הוסף אורחים"
3. הדבק את רשימת האורחים (אימייל בכל שורה):
   ```
   friend1@example.com, שם חבר 1
   friend2@example.com, שם חבר 2
   friend3@example.com
   ...
   ```
4. לחץ "הוסף"

### צעד 3: שלח לינקים (5 דקות)

**אופציה א': שליחה ידנית**
1. לחץ "תבנית הודעה" - תראה 3 הודעות מוכנות
2. לחץ "העתק" ליד כל אורח
3. שלח ב-WhatsApp/SMS

**אופציה ב': העתקה המונית**
1. לחץ "העתק הכל"
2. תקבל רשימה של כל האורחים עם הלינקים שלהם
3. העתק ושלח לכל אחד

**אופציה ג': מהטבלה**
1. לחץ "הודעה" ליד כל אורח
2. תקבל הודעה אישית עם השם שלו
3. שלח ב-WhatsApp

### צעד 4: עקוב אחרי ההתקדמות

**בזמן אמת:**
- טאב "ניהול אורחים" - רואה מי התחבר
- טאב "ניתוח מוזיקלי" - רואה את השירים המובילים
- טאב "סטטיסטיקות אורחים" (בתפריט הראשי) - סיכום כללי

**דשבורד פיילוט (אופציונלי):**
- אפשר להוסיף את `<PilotDashboard eventId={eventId} />` לכל מקום
- מציג progress bar, top 10, וכפתור ייצוא

### צעד 5: אסוף פידבק

אחרי שכולם התחברו:
1. לחץ "רענן ניתוח" בטאב ניתוח מוזיקלי
2. בדוק את top 50 השירים
3. ייצא לאקסל (אם הוספת PilotDashboard)
4. שאל את המשתתפים:
   - האם זה היה קל?
   - כמה זמן לקח?
   - האם הם ישתפו עם חברים?
   - מה אפשר לשפר?

## 📱 מה האורחים רואים

### דף נחיתה (`/guest/{token}`)
1. כותרת מושכת: "שתפו את הטעם המוזיקלי שלכם"
2. פרטי האירוע (שמות, תאריך, מקום)
3. "למה זה חשוב?" - 4 נקודות
4. "פרטיות מובטחת" - הסבר על אבטחה
5. **FAQ חדש** - 4 שאלות נפוצות
6. כפתור ענק: "התחבר עם Spotify"
7. "לוקח פחות מדקה • לא צריך Spotify Premium"

### תהליך התחברות
1. לחיצה על הכפתור → מעבר ל-Spotify
2. התחברות (אם לא מחובר)
3. אישור גישה
4. חזרה לאתר

### דף הצלחה (`/guest/{token}/success`)
1. אנימציה חגיגית ✅
2. "תודה רבה!"
3. כמה פלייליסטים וכמה שירים נאספו
4. 3 נקודות - מה קורה עכשיו
5. **קופסת שיתוף חדשה:**
   - "עזרו לנו להפוך את המסיבה למושלמת!"
   - כפתור WhatsApp
   - כפתור העתק לינק
6. "אתם יכולים לסגור את הדף הזה"
